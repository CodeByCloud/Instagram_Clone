# Instagram Clone (Flutter)

A full Instagram-like mobile app built with Flutter.
Includes:
- Authentication (Sign in / Sign up)
- Light / Dark / System theme
- Multi-language support
- Feed, posts, profile, and more
- GitHub-based deployment

## How to run
flutter pub get
flutter run
